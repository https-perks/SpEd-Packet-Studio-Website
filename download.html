Version 1.0.1

SpEd Packet Studio 1.0.1

SpEd Packet Studio is now available as a standalone Windows desktop app for creating polished special education service packets without needing separate design or PDF software.

This first public release includes a guided workflow for building student service packets from start to finish: student setup, At-a-Glance summaries, goals, data sheets, observation pages, accommodations, behavior plans, packet design, and PDF export.

Highlights

  • Create and manage local student packet projects
  • Enter student profile, school year, service areas, minutes, and case manager or team contact information
  • Build At-a-Glance summaries with editable sections
  • Create, edit, duplicate, order, and assign goals by service area
  • Add accommodations and behavior plan content
  • Create data sheets with custom columns, reusable templates, staff notes, and multiple blank table pages
  • Add observation sheets and custom note pages
  • Configure packet versions for different audiences
  • Reorder and show or hide packet pages by version
  • Choose from multiple designed packet templates
  • Customize palettes, service-area icon colors, fonts, and brand kit settings
  • Upload district or school logos for branded covers and watermarks
  • Export professional PDF packets directly from the app
  • Save project backups locally

Standalone Windows installer

The installer includes the app, backend, templates, fonts, PDF renderer, and required native PDF dependencies.

Teachers do not need to install Python, Node, Rust, GTK, WeasyPrint, or Poppler separately.

!

Notes

  • The app stores projects locally on your computer.
  • PDF export is built into the app.
  • The installer may show a Windows SmartScreen warning because this first public build is not yet code-signed.
  • Depending on your Windows settings, installation may require administrator permission.
